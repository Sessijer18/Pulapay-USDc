# Pulapay-USDc
Pulapay is a universal African money account that lets users receive, store, and send money across Mobile Money, cards, and USDC through one simple balance.
Pulapay is a universal African money account designed to abstract the complexity of payments in Africa.
It unifies Mobile Money, card payments, and USDC rails into one secure, non-custodial account experience with one balance mental model.

The platform is built to help users:

receive money from multiple sources

store value simply

send money locally and internationally without friction

Pulapay combines:

stablecoin rails powered by Coinbase

wallet infrastructure powered by Circle

a ledger-first architecture for security, reconciliation, and scalable payment routing
